# AutoIt 3 version for prototyping ww-wtlk-mltbx

### Multibox with auto mode for healer and dps in dungeons, for game "World of Warcraft: Wrath of the Lich King", version 3.3.5a (12340) Jun 24 2010

#### What's working now:
- Start wow.exe game window and auto log in to world.
- Added some hotkeys for control multibox, see below.


Hotkeys:
F1 - swith game window, HEALER -> DPS 0 -> DPS 1 -> DPS 2, if you switch to TANK game window, chain reset to start on HEALER

F2 - start/stop follow
F2 + CTRL - stop follow

F11 - /quit from game

F12 - enable/disable bot actions, for stop auto work
F12 + CTRL - disable bot actions
